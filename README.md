# Personal Homepage

This repository contains a static personal homepage for GitHub Pages.

## Local Preview

Because this is a static site, you can open `index.html` directly in a browser, or use any static server.

## Deploy to GitHub Pages

1. Push the files to the `main` branch.
2. Open repository settings.
3. Go to **Pages**.
4. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Save and wait for the deployment to complete.

## Customize Content

- Edit text structure in `index.html`.
- Adjust visual style and colors in `styles.css`.
- Update interactions in `script.js`.
